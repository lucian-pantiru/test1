Developer Express Example
How to bind MVC Scheduler to a collection of custom objects (List)

This example demonstrates how to use two List instances (resources and appointments) as MVC Scheduler data sources. The SchedulerDataObject class (which contains two properties: Appointments and Resources ) is used as a model for a corresponding "Scheduler" partial view. CRUD operations (create, update, delete) are implemented using corresponding SchedulerExtension methods (GetAppointmentsToInsert (https://documentation.devexpress.com/#AspNet/DevExpressWebMvcSchedulerExtension_GetAppointmentsToInsert[T]topic3645), GetAppointmentsToUpdate (https://documentation.devexpress.com/#AspNet/DevExpressWebMvcSchedulerExtension_GetAppointmentsToUpdate[T]topic3653), GetAppointmentsToRemove (https://documentation.devexpress.com/#AspNet/DevExpressWebMvcSchedulerExtension_GetAppointmentsToRemove[T]topic3649)) and standard List methods (Add, Remove).
